# FlowAI — Your Work, On Autopilot 🚀

A premium SaaS landing page built with React, Vite, and Framer Motion.

## 🌐 Live Demo
👉 [https://demo-site-1-eosin.vercel.app](https://demo-site-1-eosin.vercel.app)

## 📸 About This Project
FlowAI is a demo SaaS landing page designed for AI productivity tools targeting startup founders and busy professionals. Built with a premium dark theme inspired by Linear, Stripe, and Vercel.

## ✨ Features
- Modern dark theme with violet + blue + cyan gradients
- Smooth Framer Motion animations
- Fully responsive (mobile + desktop)
- 8 conversion-focused sections:
  - Hero
  - Social Proof
  - Features
  - How It Works
  - Pricing
  - Testimonials
  - CTA
  - Footer

## 🛠️ Built With
- React + Vite
- Framer Motion
- Tailwind CSS
- Deployed on Vercel

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/YOURUSERNAME/flowai-saas-demo.git

# Go into the project
cd flowai-saas-demo

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Deploy
```bash
vercel --prod
```
